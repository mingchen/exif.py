# Intro

A python utility to read / write JPEG's EXIF attributes.

# Usage

Read attributes from file:

    exif.py -r [attribute,attribute,...] file.jpg

Write attributes to file:

    exif.py -w attr1=value1,attr2=value2 file.jpg

To get help:

    exif.py -h
